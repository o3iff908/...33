# GPC Solutions - Corporate Website

A modern, professional corporate website for GPC Solutions, an engineering and consultancy company.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Multiple Sections**:
  - Hero section with navigation
  - About Us section with image gallery
  - Services showcase grid
  - How We Work process steps
  - Project gallery
  - Partners section

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Structure

```
/
├── index.html          # Main HTML file
├── style.css           # All styling
├── script.js           # Interactive functionality
├── assets/
│   ├── icons/          # Step icons
│   └── images/         # Partner images
└── *.jpg               # Various images for sections
```

## How to Run

Simply open `index.html` in a modern web browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve
```

Then navigate to `http://localhost:8000`

## Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: < 768px
